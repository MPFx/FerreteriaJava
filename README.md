# Sistema de Ferretería (Java)

## Versión
**1.0**

## Descripción
Sistema de gestión de ferretería desarrollado en Java. Permite registrar clientes, proveedores, productos (herramientas, materiales de construcción, pinturas), control de inventario, alertas de stock bajo y ventas.

## Alcance

### ✅ Qué hace
- Registrar clientes y proveedores
- Gestionar inventario por categorías (Herramienta, Construcción, Pintura)
- Control de stock y alertas de stock mínimo
- Registrar productos con características específicas
- Realizar ventas con control de stock
- Historial de ventas por cliente
- Buscar productos por código, nombre o categoría

### ❌ Qué NO hace
- No tiene interfaz gráfica (solo consola)
- No persiste datos en base de datos real

## Tipos de Producto

| Tipo | Características |
|------|-----------------|
| Herramienta | Tipo (Manual/Eléctrica/Neumática), Material, Garantía |
| Material Construcción | Unidad medida, Peso (kg), Presentación |
| Pintura | Color, Tipo, Rendimiento (m²/galón), Presentación |

## Categorías

| Categoría | Descripción |
|-----------|-------------|
| HERRAMIENTA | Martillos, taladros, llaves, etc. |
| CONSTRUCCION | Cemento, ladrillos, arena, etc. |
| PINTURA | Pinturas y accesorios |
